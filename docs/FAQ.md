# Frequently Asked Questions

## What is Dev-Docs?

Dev-Docs is a VS Code extension that uses AI to automatically generate documentation for your codebase. It can create internal docs, external docs, API docs, and more.

## How do I install Dev-Docs?

Install the Dev-Docs extension from the VS Code marketplace. Open VS Code, go to Extensions, search for "Dev-Docs", and click Install.

## How do I generate documentation with Dev-Docs?

Use the "Generate Documentation" command (Shift+Cmd+D on Mac or Shift+Ctrl+D on Windows/Linux) or right-click in your code file and select "Generate Documentation".

## What types of documentation can Dev-Docs generate?

Dev-Docs can generate internal docs, external user-facing docs, API documentation, high-level context for folders, and more.

## How do I customize the documentation generation?

Customize options in the dev-docs.json file in your project root. You can specify filters, prompts, output locations, and more.

## Can I use Dev-Docs with other tools?

Yes, Dev-Docs integrates well with tools like GitHub and Docusaurus. It can also be used alongside AI coding assistants like Continue.

## Is Dev-Docs free?

Dev-Docs offers both free and paid plans. Basic functionality is free, while advanced features may require a subscription.

## Where can I get help if I'm having issues?

Check the documentation at docs.dev or open an issue on the Dev-Docs GitHub repository if you need assistance.
